Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

JavaScript je multiplatformní, objektově orientovaný, událostmi řízený skriptovací jazyk, jehož autorem je Brendan Eich z někdejší společnosti Netscape.

JS není roven jazyku Java, oficiálně se nazývá ECMAScript. Existují různé verze, např. ES1 až ES6.

Využití:

  • Umožňuje tvořit dynamické stránky.
  • Lze vytvářet grafické aplikace včetně her.

  • Používá se k vytváření interaktivních stránek.
  • Umožňuje vytvořit webové aplikace na straně klienta (browser) i serveru (NodeJS).

  • Nabízí mnoho knihoven k nejrůznějšímu použití.

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Statements should end in a semicolon (;).
  2. JavaScript ignores multiple spaces. You can add white space to your script to make it more readable. A good practice is to put spaces around operators ( = + - * / ).
  3. Assume the type of the data that is put into the variable. The data type does not have to be explicitly defined
  4. JavaScript statements can be grouped together in code blocks, inside curly brackets {...}. The purpose of code blocks is to define statements to be executed together.
  5. JavaScript keywords are reserved words. Reserved words cannot be used as names for variables.
  6. To enter comments in the script, use "//" to start a single line comment or the combination of "/*" and "*/" to enclose a multi-line comment.

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, *=, /=, %= (modulo - zbytek po celočíselném dělení), **= (umocnění)

Logické

&&, ||, !

Porovnávací

==, === (stejná hodnota a stejný typ), != (není rovno), !== (není stejná hodnota nebo stejný typ), > (větší než), < (menší než), >= (větší nebo rovno), <= (menší nebo rovno), ? (ternární operátor)

Bitové

&, |, ~, ^, << (levý shift), >> (pravý shift), >>> (unsigned right shift)

Typové

typeof (typ proměnné), instanceof (vrátí hodnotu true, pokud je objekt instancí typu objektu)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Vypíše upozornění do „vyskakovacího“ (popup) okna */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt('Zadej sve heslo', '1235')

confirm()

/* Ve vyskakovacím okně se zobrazí otázka, kterou napíšeme do závorky za příkazem confirm a s otázky se ještě zobrazí dvě tlačítka*/
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:
Zákazníkum přijdou otravné, jsou zastaralé, jsou podezřelé.
